# Changelog

## 0.1.0

- Initial public package metadata and `minimal-agent` console script.
- Terminal agent loop with shell and structured file tools.
- Interactive REPL with session persistence, slash commands, history, diagnostics, skills, plugins, MCP, plan, workflow, and delegation helpers.
- Model gateway with fallback routes, retry controls, circuit breaker, usage ledger, and budget checks.
- Permission modes, policy file support, redacted debug bundles, and local doctor checks.
- Prompt-toolkit powered slash command and argument suggestions in interactive terminals.
